# qq-marketface-downloader
- 对QQFaceHelper的一些小修改，并添加了类型判断
- 打包下载QQ商城表情包
## 依赖
```bash
pip install requests tqdm
```

1. 复制表情详细页链接

![image](https://github.com/user-attachments/assets/ce4fef9e-103a-47d2-a8cb-f694f14861d4)
